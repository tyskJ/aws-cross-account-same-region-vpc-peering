# フォルダ構成

- フォルダ構成は以下の通り

```
.
└── envs
    ├── config
    │   ├── dummy
    │   ├── inviter-parameter.yml       INVITER スタック用パラメータファイル (Ignore Git)
    │   └── accepter-parameter.yml      ACCEPTER スタック用パラメータファイル (Ignore Git)
    ├── inviter.yml                     INVITER スタック定義ファイル
    └── accepter.yml                    ACCEPTER スタック定義ファイル
```
