動作確認手順
=====================================================================
1. EC2 にログイン
---------------------------------------------------------------------
* `envs` フォルダ配下でデプロイ実施後、 `Outputs` にて `KeyPair作成コマンド` 及び `SSHコマンド` が出力される
* リクエスタ側の `KeyPair作成コマンド` を実施し、 `envs` フォルダ配下に `KeyPair` を作成
* その後、 `SSHコマンド` を実行し リクエスタ側の EC2 にログインする

2. `ping` 実行
---------------------------------------------------------------------
.. code-block:: zsh

    ping アクセプタ側EC2インスタンスプライベートIPアドレス
